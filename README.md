# 🌅 Amanecer-CPM

Plataforma de alquiler de propiedades en Colombia.

## Instalación

```bash
git clone https://github.com/Darwinlopez-mrY/amanecer-CPM.git
cd amanecer-CPM
git checkout entrega
Configuración
Backend:

bash
cd backend
cp .env.example .env
npm install
Frontend:

bash
cd frontend
cp .env.example .env
npm install
Ejecución
Backend:

bash
cd backend
npm run dev
Frontend (otra terminal):

bash
cd frontend
npm run dev
